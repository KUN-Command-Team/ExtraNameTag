### 概要
エンティティのネームタグ表示を拡張し、通常の名前の上に追加の文字列を複数行表示できるMODです。

ミニゲームで役割表示を増やしたいとき、プレイヤーやモブの状態ラベルを見やすくしたいときに便利です。クライアントとサーバーの両方に導入してください。

### 利用方法
- 名前を追加します
   ```bash
   /nametag <targets> push <name>
   ```
- 直近に追加した名前を1つ削除します
   ```bash
   /nametag <targets> pop
   ```
- 追加した名前をすべて削除します
   ```bash
   /nametag <targets> clear
   ```
